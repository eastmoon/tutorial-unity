※ IDE path using in Unity 5

1. Setting player information
>> Edit -> Project setting -> Player
>> File -> Build settings -> Player settings

2. Choose platforms
>> File -> Build settings ( Ctrl + Shift + B )-> platform ( choose one ) -> Switch platform

3. Build
>> File -> Build & Run ( Ctrl + B )

◎ Build Andorid app 

1. Setting icon

2. Identifier
>> Other Settings -> Bundle Identifier -> type identifier

3. Minimum API Level
>> Other Settings -> Minimum API Level -> Chosse min API Level ( suggest Android 4.0, API level 14 )

4. Install Location
>> Other Settings -> Install Location -> Choose "Automatic"

5. Orientation
>> Resolution and Presentation -> Default Orientation -> Chosse screen orientation

6. Publishing Settings
When app update to android market, setting publishing key.
